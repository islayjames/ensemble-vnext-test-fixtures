## API Structure

```
Base URL: http://localhost:3001/api

/boards                    - Board CRUD
/boards/:boardId/columns   - Column CRUD (nested under board)
/columns/:id               - Column update/delete
/columns/:columnId/cards   - Card creation
/cards/:id                 - Card CRUD
/cards/:id/move            - Card repositioning
/reports/tasks             - Task reporting
/reports/metrics           - Aggregated metrics
/reports/export            - CSV export
```
